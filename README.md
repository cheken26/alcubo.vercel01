- **Logo:** alcubo³ (superíndice) + ARQUITECTOS
- **Paleta:** negro `#0a0a0a` · blanco `#ffffff` · gris cards `#1a1a1a` · gris sección `#111111`

## 📁 Estructura

```
index.html      — Sitio completo (HTML + CSS + JS embebidos)
README.md       — Documentación del proyecto
```

---

## 📂 Secciones

1. **Hero** — Imagen fullscreen, título, CTA "Ver proyectos"
2. **Proyectos** — Grid masonry 3 col (desktop) / 1 col (móvil) — 8 proyectos con filtros
3. **Estudio** — Descripción del equipo + estadísticas + imagen
4. **Servicios** — 4 cards: Residencial, Comercial, Interiorismo, Urbanismo
5. **Contacto** — Formulario + datos de contacto + imagen
6. **Footer** — Logo, links, Instagram, copyright

---

## 🔧 Tecnologías

- HTML5 semántico
- CSS3 (custom properties, grid, columns masonry, transitions)
- JavaScript vanilla (IntersectionObserver, hamburger, filtros, smooth scroll)
- Google Fonts: Space Grotesk + Inter
- Imágenes: Unsplash (CDN)
